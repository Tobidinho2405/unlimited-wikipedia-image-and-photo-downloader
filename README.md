## 🧹 Troubleshooting

- **App won’t start:** Make sure Python 3.8+ is installed and added to your system PATH.
- **No images download:** Verify that you entered a valid Wikipedia article name or full URL.
- **Permission errors:** Run the Command Prompt or Terminal as administrator or with elevated rights.
- **Connection errors:** Check your internet connection and try again.
- **Images save empty or corrupted:** Some Wikipedia images might be restricted or removed; try another article.

If you still have issues, please create a new issue on the GitHub repository page.
